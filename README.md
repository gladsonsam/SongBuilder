# SongBuilder

A powerful web application for managing and formatting song lyrics with chords. Supports multiple formats including Ultimate Guitar and Freeshow.

## Features

- Import songs from Ultimate Guitar format
- Export to Freeshow format
- Visual chord editor
- Multiple section support (Verse, Chorus, Bridge, etc.)
- Precise chord alignment with lyrics

## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone https://github.com/YOUR_USERNAME/songbuilder.git
```

2. Install dependencies:
```bash
cd songbuilder
npm install
```

3. Start the development server:
```bash
npm start
```

## Usage

1. Import a song in Ultimate Guitar format
2. Edit chord positions visually
3. Export to your desired format (Freeshow, etc.)
